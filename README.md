# Country Intelligence Dashboard

A production-style country comparison dashboard built with Next.js App Router.

The application compares two countries and aggregates:
- geographic information
- weather conditions
- environmental insights
- operational comparison metrics

Users can compare countries in a unified interface instead of manually searching multiple services separately.

---

# Features

- Country comparison
- Live weather comparison
- Computed environmental insights
- Graceful partial failure handling
- Typed API contracts
- Backend API orchestration
- Responsive modern dashboard UI

---

# Tech Stack

## Frontend
- Next.js App Router
- React
- TailwindCSS
- TypeScript

## Backend
- Next.js Route Handlers
- REST Countries API
- OpenWeather API

# Environment Variables

Create a `.env.local` file in the project root:

# env
OPENWEATHER_API_KEY=your_openweather_api_key
to setup openweather api key go to https://openweathermap.org/ sigin open profile and select my api key. generate an api key and paste it in .env file

First, run the development server:
